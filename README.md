# E-Commerce UI Design with React

https://arnold-dev-react-ui.herokuapp.com/cart

## Set-up

- Clone the repo
``` git clone https://github.com/Arnold-git/React-E-commerce-App-UI-Design ```

- Install Package
``` npm install package.json```

- Test Locally
``` npm start ```
